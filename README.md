# AutoSchedule

> 팀 단위 회의·일정·작업시간을 AI가 자동으로 최적화해주는 협업 스케줄 관리 시스템  
> 핵심 기능: **실시간 협업 + 자동 스케줄링**

---

## 주요 기능
- 사용자 관리 (회원가입, 로그인, 팀 초대)
- 일정 입력 및 우선순위 기반 자동 스케줄링
- WebSocket 실시간 협업
- FullCalendar 시각화
- Flyway 기반 DB 버전 관리
